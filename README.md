# MagicalHighSchool3
입시정보 어플리케이션

개요
---
for 곧3들과 고3들을 위한 공익성 어플리케이션
입시정보를 대학별로 한눈에 쉽게보고 지원자, 경쟁률, 최대내신, 최소내신, 등록금, 면접일자, 지원일자 등의 정보를 확인할 수 있습니다.
<br>
또한 입시생들끼리의 커뮤니티를 형성하고 수능 전략등을 공유하며, 대학생 멘토와의 시간도 갖고 대학생 멘토는 자신의 과외등의 정보를 홍보할 수 있습니다.
